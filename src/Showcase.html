Core Architecture

Layer-based design with Entity-Component composition

Layer System

You can add as many layers as you want

OnAttach()

Initialize resources

OnUpdate()

Frame logic

OnInterface..()

Render UI

OnInput()

Handle events

OnDetach()

Cleanup

Project Orchestration

The Project singleton manages:

  • Main game loop and delta time
  • Layer stack management
  • Automatic layer lifecycle execution
  • Editor and runtime integration

Layer Implementation Example


class GameLayer : public Bonfire::Layer 
{
public:
    virtual void OnAttach() override 
    {
        // first frame
    }

    virtual void OnUpdate(Bonfire::DeltaTime delta_time) override 
    {
        // gameplay
    }

    virtual void OnInterfaceUpdate() override 
    {
        // game interface
    }

    virtual void OnInput(Input& input) override
    {
       // handle player input
    }
};
