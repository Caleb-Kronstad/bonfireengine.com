Build Requirements

System Requirements

  • OS: Windows 10/11 (x64 only) - Linux and MacOS support planned for later date
  • GPU: OpenGL 4.6 compatible graphics card
  • Disk Space: ~500MB

Required Tools

  • Any C++ Compatibile IDE on Windows - JetBrains Rider or Visual Studio 2022 recommended
  • C++20 compiler (MSVC)
  • Windows SDK (latest)

Dependencies (Included)

  • GLFW - Windowing and input
  • GLAD - OpenGL loader (4.6 Core)
  • ImGui - UI framework
  • ImGuizmo - 3D manipulation gizmos
  • GLM - Math library
  • Assimp - 3D model loading
  • stb_image - Image loading
  • Jolt - Physics and collision
  • nlohmann/json - Serialization

Build Instructions

  1. Clone the repository with submodules

    git clone --recursive https://github.com/Caleb-Kronstad/Bonfire Bonfire
    cd Bonfire
    Note: The --recursive flag automatically downloads all dependencies (~200MB)

  2. Run the setup script

    cd Scripts
    Setup-Windows.bat
    Note: This runs Premake5 and generates Bonfire.sln in the root directory

Creating Custom Projects

  1. Copy Bonfire/ folder and rename
  2. Modify Source/ProjectControl.cpp to add custom layers
  3. Inherit from Bonfire::Layer class
  4. Re-run Scripts/Setup-Windows.bat
